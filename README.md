
# PrimeNG Tutorial Project

This is a tutorial project demonstrating the usage of **PrimeNG** components in an **Angular** application. The project covers basic integration, UI components, forms, and data tables.

## Project Overview

This project is intended as a learning resource to understand how to:

- Integrate PrimeNG in Angular
- Use PrimeNG UI components like buttons, tables, forms, dialogs
- Apply theming and responsive design
- Handle user interactions and data binding

## Prerequisites

Make sure you have the following installed:

- **Node.js** >= 18.x
- **Angular CLI** >= 18.x
- **npm** or **yarn**

Verify installation:

```bash
node -v
npm -v
ng version
